# ES

Acá estan los dos proyectos de Técnicas Digitales 3 que hice. El primero es un sistema operativo implementado en C y ASM para un Cortex A8. El mismo implementa distintos features interesantes. El más interesante es la capacidad de ejecutar tareas de manera dinámica (código de tareas que puede ser suministrado en tiempo real). Para más detalles sobre el desarrollo, dirigirse al pdf titulado _"PPT - OS Explained"_.

El segundo proyecto es un servidor escrito en C que publica las lecturas de un sensor a quienes se conecten. No es nada muy loco.

# EN
Here are the two projects from Digital Design 3 that I made. The first one is an operating system implemented in C and ASM for a Cortex A8. It implements several interesting features. The most interesting one is the ability to execute tasks dynamically (new task code that can be supplied in real time). For more details on the development, refer to the PDF titled _"PPT - OS Explained"_.

The second project is a server written in C that publishes sensor readings to whoever connects to it. Nothing too crazy.
